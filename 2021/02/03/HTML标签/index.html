  总结一些常用的 HTML 标签和 CSS 写法

HTML

块元素

块元素的特征：

  • 总是从新的一行开始,并且独占一行

  • 高度、宽度都是可控的

  • 宽度没有设置时，默认为 100%

  • 块级元素中可以包含块级元素和行内元素

行内元素

  • 不会独占一行,行内元素会和行内元素在同一行排列

  • 不能设置宽度和高度属性

  • 宽度没有设置时，默认为 100%

  • CSS 设置竖直方向的 margin 属性无效

  • 默认宽度为该元素包含内容的宽度

行内块元素

  • 行内块元素和块元素一样具有宽高属性
  • 默认的宽度和行内元素一样,以元素内包含的内容作为默认宽度
  • 和行内元素一样，不会另起一行,能排列在其他行内或者行内块元素后面
