css 简介

CSS 指层叠样式表 (Cascading Style Sheets)，HTML都有自己的默认样式，但是很单一不能满足需求，所以需要css样式表来自定义标签的样式。

优先级

css中可能会存在样式冲突，所以需要了解css样式的优先级
1.选择器的优先级
行内样式<元素选择器<id选择器
2.如果在选择器层面处于相同优先级,则看谁描述的详细,越详细的优先级越高
.main .child .active{}的优先级肯定大于.child .active{} 因为他对要选择的元素描述的更详细
同时也要配合上面的选择器优先级判断
.main #child{}的优先级大于  .main .child .active{} 因为在第二层id选择器的优先级就高于class类选择器了
3.最高优先级
.main{color:red; !important} 加上!important后 元素的样式可以摆脱前面这些规则，变成优先级最高的样式

盒模型

CSS 盒模型本质上是一个盒子，封装周围的 HTML 元素，它包括：边距，边框，填充，和实际内容。

  • margin 外边距

    标签添加外边距属性后，会与周围的标签产生间距(给定的margin值)，如果两个上下相邻的元素，上面的元素给定margin-bottom，下面的元素设定margin-top，不会同时生效，只会取其中较大的值，比如说上元素margin-bottom:10px,比如说下元素margin-bottom:10px，两个元素之间的间距不会变成10+10=20px,而是10px,如果上为20px，下为10px，则间距取最大值20px；左右间距可以同时生效，不存在抵消。

  • border 边框

    可以设定元素的边框,边框是以padding区域为边界绘制

  • padding 内边距

    可以通过padding设置标签的内边距，添加此属性后盒子会被撑大(因为添加了内边距)，同时content的位置与border的位置也有了间距(给定的padding值)，如果元素的box-sizing属性为border-box，添加padding属性会不会增加盒子的区域而是缩小content的区域(盒子往内收缩)

  • content 盒子的文本区域

    盒模型中的视图区域
